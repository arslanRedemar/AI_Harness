```

║                                                                          ║

║  │   프로토타입 / 설계 문서 / 현재 논의 중인 화면                     │    ║
║  └─────────────────────────────────────────────────────────────────┘    ║
║                                                                          ║
║                        🎙️  마이크 (전방향)                               ║
║                                                                          ║
║    ── ── ── ── ── ── ── ── 테이블 ── ── ── ── ── ── ── ──               ║
║   |                                                         |            ║
║   |  👤 고객 A   👤 고객 B   👤 고객 C                       |            ║
║   |  (기획팀장)  (개발팀장)  (대표)                            |            ║
║   |                                                         |            ║
║   |  👤 FDE 1         👤 FDE 2                              |            ║
║   |  (진행·발표)       (기록·관찰)                            |            ║
║   |                    💻 에이전트 화면                       |            ║
║    ── ── ── ── ── ── ── ── ── ── ── ── ── ── ── ── ── ──               ║
║                                                                          ║
║   📡 에이전트 서버 연결 (클라우드 / 온프레미스)                            ║
╚══════════════════════════════════════════════════════════════════════════╝
```

- **FDE 1**: 고객과 직접 대화하며 미팅을 이끈다. 에이전트 화면을 직접 보지 않는다
- **FDE 2**: 에이전트 화면을 모니터링하며 중요한 서제스천을 FDE 1에게 조용히 전달한다
- **마이크**: 전방향 마이크가 대화를 실시간 수음 → STT → 에이전트로 전달
- **에이전트 화면**: FDE 2의 노트북. 고객에게는 보이지 않는다

## 8. 서제스천 출처 표시 방식

FDE가 서제스천의 신뢰도를 즉각 판단할 수 있도록 출처를 아이콘으로 구분한다.

```
┌─────────────────────────────────────────────────────────────────────────┐
│  💡 SUGGESTIONS                                                          │
│                                                                          │
│  📚 [Domain KB / 위치정보법_2024.pdf]                      신뢰도: 높음  │
│     위치 정보 수집 시 사전 동의 필수 (위치정보법 제15조)                  │
│     → 기사 앱 설계 시 동의 팝업 시점 명시 필요                            │
│                                                                          │
│  🔍 [웹 검색 / 2024.11 기사]                               신뢰도: 중간  │
│     국내 배달앱 평균 응답속도 200~300ms                                   │
│     → 수치 검증 후 SLA에 반영 권장                                        │
│                                                                          │
│  💬 [3/15 미팅 기록 19:22]                                 신뢰도: 확정  │
│     "앱 내 결제는 1차 스펙 제외" — 고객 대표 합의                         │
│     → 현재 발언과 충돌 감지                                               │
└─────────────────────────────────────────────────────────────────────────┘
```
